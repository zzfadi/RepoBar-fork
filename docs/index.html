RepoBar
macOS menu bar intelligence for GitHub

Stay ahead of your repositories without opening a browser.

RepoBar is a focused macOS menu bar app that surfaces the signal you care about: CI status, issues and PR counts, releases, traffic, and a compact activity heatmap for every repo you pin.

  • 01 Real-time CI state with fast jump to Actions or Checks.
  • 02 Releases, activity, and traffic in a single glanceable grid.
  • 03 Built for power users: pin, sort, and refresh on your schedule.
Download latest GitHub
Install via Homebrew
brew install --cask steipete/tap/repobar
Update anytime with brew upgrade.
